# Проект: Цветовая Легенда на Next.js и shadcn/ui

## Описание

### Проект состоит из двух основных компонентов:
 1. HomePage (DataInputHandler): Главный экран, содержащий текстовое поле ввода (<textarea>), где пользователь может ввести числа, разделенные пробелами, запятыми или переносами строк. Компонент парсит ввод, игнорирует нечисловые значения и отображает статистику (количество валидных/отброшенных токенов).
 2. ColorLegend: Компонент легенды, который принимает массив чисел и строит горизонтальную шкалу из N одинаковых элементов (где N — количество уникальных валидных значений). Цвет каждого элемента определяется по линейному градиенту зеленый (min) → красный (max). При совпадении min == max используется однородный средний (желтый/оранжевый) цвет. При наведении на элемент отображается его точное значение.

## Используемые технологии:
 * Next.js 14 (App Router)
 * TypeScript
 * React
 * Tailwind CSS
 * shadcn/ui (компоненты Card, Label, Textarea, Badge)

## Установка и запуск
 Следуйте этим инструкциям, чтобы развернуть проект локально.

### Требования
 Убедитесь, что у вас установлен Node.js (рекомендуется версия 18.17.0 или новее) и NPM.

### Пошаговая инструкция:

 1. Клонируйте репозиторий (если вы используете Git) или создайте новый проект Next.js, как описано в предыдущих шагах:

   #### bash
    ```npx create-next-app@latest```
    (Выберите TypeScript, ESLint, Tailwind CSS, src/ directory, App Router).

 2. Перейдите в директорию проекта:

  #### bash
   ```cd [имя_вашего_проекта]```
   
 3. Установите и настройте shadcn/ui:
  
  #### bash
  ```npx shadcn-ui@latest init```
  ```npx shadcn-ui@latest add card label textarea badge```
 4. Запустите проект в режиме разработки:

  #### bash
  ```npm run dev```
 5. Откройте приложение в браузере:http://localhost:3000

### Использование
Введите набор чисел в текстовое поле. Вы можете использовать пробелы, запятые или новые строки в качестве разделителей. Приложение автоматически обновит легенду и счетчики валидных/отброшенных значений. Наведите курсор на любой из цветных сегментов, чтобы увидеть точное значение, которое он представляет.
